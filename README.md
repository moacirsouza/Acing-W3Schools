# Acing W3Shools

Conheci o [W3Schools](https://latex-tutorial.com) quando eu era adolescente (no
final da dédaca de 1990) e se for pensar bem, ele foi um dos primeiros contatos
que tive com código (talvez **O** primeiro), antes de entrar na faculdade.

Neste projeto, eu registro os esforços para finalmente "dominar" o _site_,
completando todos os tutorias disponíveis por lá.

O histórico é mantido em uma tabela, cujo formato é definido pelas **Regras
para Registros de Evolução de Projetos**
([R.R.E.P.](https://github.com/moacirsouza/rrep)).
